# VoyageOS

A level-based voyage arcade game built for **[Remix](https://remix.gg)**.

🎮 **[Play it on Remix »](https://remix.gg/g/81cd5619-b07e-4560-9f19-9754db94a452)**

## About
Chart a course through hand-built levels and rack up a high score. Mobile-first, portrait.

## Tech
- HTML5 game with a custom in-repo level editor (`tools/level-editor.html`)
- Integrated with the Remix SDK (leaderboard, game-over, mute)
- Assets hosted on Remix blob storage
